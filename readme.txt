Lightweight autoposter to publish WordPress content to selected social media platforms with short link support.

== Description ==

HashPoster is a lightweight, powerful WordPress plugin that automatically shares your posts to multiple social media platforms with customizable content and media integration.

### Features

* **Multiple Platforms Support**: X (Twitter), Facebook, LinkedIn (business pages), Bluesky, and Reddit.
* **Automatic Open Graph & Twitter Cards**: Optimizes your content for social sharing automatically.
* **Media Sharing**: Attaches featured images as cards or uploads them directly to social platforms.
* **URL Shortening**: Bitly integration and WordPress shortlinks support.
* **Flexible Post Templates**: Customize your social media posts with various templates tags.
* **Scheduling**: Delay posting to avoid flooding your followers' feeds.
* **Secure Credential Management**: All API keys and tokens are stored securely.
* **Developer Friendly**: Well-documented code with filters for customization.

== Installation ==

### GitHub Installation
1. Download the latest release zip file from GitHub.
2. Log in to your WordPress dashboard.
3. Go to Plugins > Add New > Upload Plugin.
4. Choose the downloaded zip file and click "Install Now".
5. Activate the plugin.

Once installed, future updates will be handled automatically via the "Updates" dashboard!

== FAQ ==

= Posts are not showing up on social platforms? =
Check your API credentials for each platform under the HashPoster settings. Ensure your tokens haven't expired and verify that your app has Write permissions for each platform.

= Featured images are not appearing? =
Make sure your WordPress posts have featured images set. Also, ensure your images meet platform requirements for size and format.

= Shortlinks are not generating? =
Verify that your Bitly token is valid and active in the plugin settings, or make sure your WordPress shortlink functionality is enabled.
